# Sobre este curso

Este curso oferece uma cobertura abrangente do Protocolo de Contexto de Modelo (MCP), com foco na criação de servidores e clientes MCP usando o SDK Python. Você aprenderá sobre os três elementos básicos do MCP — ferramentas, recursos e prompts — e entenderá como eles se integram ao Claude AI para criar aplicações poderosas sem escrever código de integração extenso.

## O que você aprenderá

* Compreenda a arquitetura MCP e como ela transfere a definição e a execução de ferramentas do seu servidor para servidores MCP especializados.
* Saiba mais sobre o sistema de comunicação agnóstico ao transporte do MCP e os tipos de mensagens usados ​​entre clientes e servidores.
* Explore o fluxo completo de requisição e resposta, desde as consultas do usuário, passando pelos clientes MCP, até os serviços externos e de volta ao Claude.
* Crie servidores MCP usando o SDK Python com decoradores para definir ferramentas em vez de escrever esquemas JSON manualmente.
* Implemente a funcionalidade de gerenciamento de documentos com ferramentas para leitura e edição de documentos, utilizando descrições de campos e dicas de tipo.
* Utilize o Inspetor de Servidor MCP integrado para testar e depurar a funcionalidade do seu servidor em uma interface baseada em navegador.
* Defina recursos para expor dados somente leitura, incluindo recursos diretos com URIs estáticas e recursos com modelos e parâmetros.
* Implementar funcionalidade de leitura de recursos nos clientes com tratamento adequado de tipos MIME para conteúdo JSON e de texto.
* Crie instruções pré-elaboradas e de alta qualidade para fluxos de trabalho comuns, como formatação de documentos.
* Entenda quando usar cada primitiva do MCP: ferramentas (controladas pelo modelo), recursos (controlados pelo aplicativo) e prompts (controlados pelo usuário).
* Analise padrões práticos de integração, incluindo funcionalidade de preenchimento automático e injeção de contexto para conversas com IA.

## Pré-requisitos
* Conhecimento prático de programação em Python
* Conhecimento básico de JSON e padrões de requisição-resposta HTTP.

## A quem se destina este curso?
* Desenvolvedores que desejam criar servidores MCP
